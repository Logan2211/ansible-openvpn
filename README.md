ansible-openvpn
=========

An ansible role to install and configure OpenVPN.

[![Build Status](https://travis-ci.org/logan2211/ansible-openvpn.svg?branch=master)](https://travis-ci.org/logan2211/ansible-openvpn)

Requirements
------------

Ubuntu 16.04 or RHEL/CentOS 7

Example Playbook
----------------

    - hosts: openvpn
      roles:
         - { role: openvpn }
